# Contributing to Agentic Platform

Thank you for your interest in contributing!

## How to Contribute

1. **Fork the repository** and clone it locally.
2. **Create a new branch** for your feature or bugfix:
    ```bash
    git checkout -b my-feature
    ```
3. **Write clear, concise code** and add/update tests as appropriate.
4. **Commit your changes** with meaningful messages.
5. **Push to your fork** and open a Pull Request (PR) against the `main` branch.

## Code Guidelines

- Keep code modular and place it in the relevant sub-package.
- Add docstrings to public methods and classes.
- Add new installs in req.txt
- Write descriptive commit messages.

## Reporting Issues

If you find a bug or want to suggest a feature, please open an issue with details and steps to reproduce (if applicable).

## Code of Conduct

Be respectful in all interactions and discussions.
